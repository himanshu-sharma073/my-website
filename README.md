# Himanshu Sharma — Lead Platform Engineer Portfolio

This repository hosts my personal portfolio: a single-page, high-impact profile for a Lead Platform Engineer focused on scalable architecture, resilient microservices, and technical leadership.

## Highlights

- **Hero & Systems Snapshot** – executive summary, platform metrics (30+ orchestrated workflows, 7 microservices, 99.9% reliability), and instant access to resume/contact links.
- **GitHub Contributions** – live activity widget plus narrative describing my open-source focus areas (Kogito tooling, Elasticsearch, Mongoose, coding challenges).
- **Skills Matrix** – categorized toolkit covering runtimes, frameworks, cloud/devops, databases, tooling, and architectural patterns.
- **Experience Timeline** – detailed case studies from Freshworks (iPaaS modernization, CDC migration, omnichannel services, CI/CD automation) and PS QuickIT (team leadership, CRM delivery, talent development).
- **Awards & Education** – recognition from Freshworks and PS QuickIT hackathons plus AKTU / IIES academic performance.
- **Contact Section** – website, phone, email, LinkedIn, HackerRank, and a downloadable PDF resume.

## Tech Stack

- Static assets only: `index.html`, `styles.css`, `script.js`
- Typography: Google Fonts (`Inter`, `Space Grotesk`)
- Animations: Intersection Observer (scroll reveal + sticky navigation highlights)
- Assets:
  - Resume: `assets/himanshu-sharma-resume.pdf`
  - Favicon: `assets/favicon.svg`
  - Open Graph preview: `assets/og-cover.svg`
  - GitHub contribution chart: [`ghchart.rshah.org`](https://ghchart.rshah.org/)

## Run Locally

```bash
git clone https://github.com/himanshu-sharma073/my-website.git
cd my-website

# Option 1: open directly
start index.html        # Windows
open index.html         # macOS

# Option 2: lightweight server
npx serve .             # or python -m http.server 8080
```
